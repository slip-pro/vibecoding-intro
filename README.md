# Vibe-coding Intro

**Твой первый сайт за один вечер.**
Пошаговый путь от нуля до задеплоенного сайта
с помощью AI.

## Что это

Vibecoding Intro -- открытый образовательный проект от команды KKTS.AI,
который проведёт тебя от пустого экрана до работающего
сайта в интернете. Никакого предварительного опыта
в программировании не нужно.

Всю техническую работу делает AI-помощник
(GitHub Copilot, встроенный в VS Code) -- ты описываешь,
какой сайт хочешь, а он пишет код, настраивает окружение
и объясняет каждый шаг простым языком.

> Результат -- не учебная поделка, а настоящий сайт
на твоём домене с HTTPS, который видит весь мир.

## Для кого

- Ты никогда не программировал, но хочешь попробовать
- Тебе нужен сайт, а платить агентству -- дорого
- Ты предприниматель, фрилансер или блогер, которому
  нужна своя страница в интернете
- Тебе просто интересно, как работает веб-разработка

## Что ты получишь

- Работающий сайт-лендинг на своём домене
- Понимание, как устроена веб-разработка изнутри
- Навык работы с AI-инструментами (GitHub Copilot)
- Готовую инфраструктуру: Git, Docker, VPS

## Как это работает

Ты ставишь VS Code, клонируешь этот репозиторий
и пишешь "Привет" в GitHub Copilot. AI определяет,
на каком ты этапе, и ведёт дальше -- шаг за шагом.

Путь состоит из фаз:

1. **Знакомство** -- рассказываешь, какой сайт хочешь
2. **Node.js** -- ставим движок для сайта
3. **Создание проекта** -- запускаем сайт локально
4. **Разработка** -- делаем лендинг по твоему описанию
5. **Git и GitHub** -- сохраняем код в облаке
6. **Docker** -- упаковываем сайт для переезда
7. **Домен и VPS** -- покупаем адрес и сервер
8. **Деплой** -- запускаем сайт в интернете

## Стоимость

| Что | Цена | Комментарий |
|-----|------|-------------|
| VS Code | Бесплатно | Редактор кода |
| Copilot Free | Бесплатно | 50 запросов/мес, модель Claude Haiku |
| Copilot Pro | ~$10/мес | 300 запросов/мес, модели Claude Sonnet и Opus |
| Домен | ~800 руб/год | Адрес сайта (Selectel) |
| VPS | ~500 руб/мес | Сервер (Selectel) |

> **Про модели:** в бесплатном Copilot доступен Claude Haiku --
> быстрая и лёгкая модель. В Pro открываются Claude Sonnet
> (отличный баланс качества и скорости) и Claude Opus
> (самая мощная, глубже понимает контекст и лучше менторит).
> [Подробнее о моделях](docs/01-setup.md#шаг-221-зачем-нужен-copilot-pro)

> **Другие инструменты:** курс написан для VS Code + Copilot,
> но вайб-кодинг работает и с другими AI-редакторами:
> [Cursor](https://cursor.com),
> [Windsurf](https://windsurf.com),
> [Google Antigravity](https://developers.googleblog.com/build-with-google-antigravity-our-new-agentic-development-platform/),
> [Claude Code](https://docs.anthropic.com/en/docs/claude-code),
> [OpenCode](https://opencode.ai).
> [Подробнее](docs/00-intro.md#другие-ai-инструменты)
