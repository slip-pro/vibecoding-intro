# Vibe-coding Intro

**Твой первый сайт за один вечер.**
Пошаговый путь от нуля до задеплоенного сайта
с помощью AI.

## Что это

Vibecoding Intro -- открытый образовательный проект от команды [KKTS.AI](https://kkts.ai),
который проведёт тебя от пустого экрана до работающего
сайта в интернете. Никакого предварительного опыта
в программировании не нужно.

Всю техническую работу делает AI-помощник
(встроен в редактор [Cursor](https://cursor.com)) --
ты описываешь, какой сайт хочешь, а он пишет код,
настраивает окружение и объясняет каждый шаг простым языком.

> Результат -- не учебная поделка, а настоящий сайт
> на твоём домене с HTTPS, который видит весь мир.

## Как это работает

Ты ставишь Cursor, клонируешь этот репозиторий
и пишешь "Привет" в AI-чате. Cursor определяет,
на каком ты этапе, и ведёт дальше -- шаг за шагом.

Путь состоит из фаз:

1. **Знакомство** -- рассказываешь, какой сайт хочешь
2. **Node.js** -- ставим движок для сайта
3. **Создание проекта** -- запускаем сайт локально
4. **Разработка** -- делаем лендинг по твоему описанию
5. **Git и GitHub** -- сохраняем код в облаке
6. **Docker** -- упаковываем сайт для переезда
7. **Домен и VPS** -- покупаем адрес и сервер
8. **Деплой** -- запускаем сайт в интернете


## Что ты получишь

- Работающий сайт-лендинг на своём домене
- Понимание, как устроена веб-разработка изнутри
- Навык работы с AI-инструментами (Cursor)
- Готовую инфраструктуру: Git, Docker, VPS


## Стоимость

| Что | Цена | Комментарий |
|-----|------|-------------|
| Cursor | Бесплатно | Редактор кода с AI внутри |
| Cursor Pro | ~$20/мес | Больше запросов, выбор модели (Claude, GPT) |
| Домен | ~800 руб/год | Адрес сайта (Selectel) |
| VPS | ~500 руб/мес | Сервер (Selectel) |

> **Про модели:** в бесплатном Cursor AI сам выбирает
> лучшую модель (Auto mode). В Pro можно выбрать вручную:
> Claude Sonnet, Claude Opus, GPT и другие.
> [Подробнее](docs/01-setup.md#шаг-22-cursor-free-vs-pro)

> **Другие инструменты:** курс написан для Cursor,
> но вайб-кодинг работает в трёх "пространствах": в браузере
> (v0, Lovable, Replit), в терминале (Claude Code, Aider)
> и в IDE (VS Code + Copilot, Windsurf, Antigravity).
> [Подробнее](docs/00-intro.md#где-живёт-вайб-кодинг)
