# Установка и первый запуск

> Всего 3 шага -- и AI-помощник начнёт помогать

**⏱ Время: ~15 минут**

Тебе нужно установить всего две вещи: редактор кода и AI-помощника.
Всё остальное (Node.js, Git, Docker) Copilot поможет поставить потом,
когда это понадобится. Не нужно ставить всё заранее.

---

## Шаг 1: Установка VS Code

**⏱ ~5 минут**

### Что это такое

VS Code -- это бесплатный редактор кода от Microsoft. Думай о нём
как о Word, только для кода. Здесь ты будешь видеть свой проект
и общаться с AI-помощником.

### Установка

1. Открой в браузере сайт [code.visualstudio.com](https://code.visualstudio.com)
2. Нажми большую синюю кнопку **"Download for Windows"**
3. Дождись окончания скачивания (файл появится внизу браузера)
4. Открой скачанный файл двойным кликом
5. Появится окно установки. Нажми **"Далее"**
6. Прими лицензионное соглашение. Нажми **"Далее"**
7. На экране "Выберите дополнительные задачи":
   - Поставь галочку **"Add to PATH"** (добавить в PATH) -- это важно
   - Остальные галочки можно оставить как есть
8. Нажми **"Установить"**
9. Дождись окончания установки
10. Нажми **"Готово"**

<!-- screenshot: сайт VS Code с кнопкой Download -->
<!-- screenshot: экран установки с галочкой Add to PATH -->

### Первый запуск

1. Найди VS Code в меню "Пуск" (набери "VS Code" в поиске Windows)
2. Запусти его
3. Ты увидишь приветственную страницу -- это нормально, просто закрой её

<!-- screenshot: приветственная страница VS Code -->

> **⚠️ Если что-то пошло не так**
>
> - **VS Code не запускается** -- перезагрузи компьютер и попробуй снова
> - **Ошибка при установке** -- нажми правой кнопкой мыши на установщик,
>   выбери "Запуск от имени администратора"
> - **Не находится в меню "Пуск"** -- попробуй набрать полное название
>   "Visual Studio Code" (не путай с Visual Studio -- это другая программа)

### Русификация (необязательно)

Если хочешь, чтобы интерфейс VS Code был на русском:

1. Нажми **Ctrl+Shift+X** -- откроется панель расширений слева
2. В строке поиска набери **"Russian Language Pack"**
3. Найди расширение от Microsoft (с иконкой флага)
4. Нажми кнопку **"Install"** (Установить)
5. Появится уведомление внизу справа -- нажми **"Change Language and Restart"**
6. VS Code перезапустится на русском языке

<!-- screenshot: установка Russian Language Pack -->

> Это необязательный шаг. Многие разработчики работают с английским
> интерфейсом -- большинство инструкций и туториалов написаны
> на английском. Но если с русским комфортнее -- ставь без сомнений.

## Шаг 2: Настройка GitHub Copilot

**⏱ ~5 минут**

### Что это такое

GitHub Copilot -- это AI-помощник, встроенный прямо в VS Code.
Ты пишешь ему что хочешь на обычном русском языке -- он делает.
Именно он будет вести тебя через весь процесс создания сайта.

В отличие от многих AI-сервисов, Copilot работает без VPN
и доступен из России через VS Code.

### Шаг 2.1: Регистрация на GitHub

GitHub -- это сайт, где разработчики хранят свой код. Аккаунт
на GitHub нужен для Copilot (и пригодится позже, когда будем
сохранять твой сайт).

1. Открой в браузере [github.com](https://github.com)
2. Нажми **"Sign up"** (Зарегистрироваться)
3. Введи email, придумай пароль и имя пользователя (username)
4. Пройди проверку (решение головоломки)
5. Подтверди email -- зайди в почту и нажми на ссылку из письма GitHub
6. Готово, ты на GitHub!

<!-- screenshot: страница регистрации GitHub -->
<!-- screenshot: письмо подтверждения email от GitHub -->

### Шаг 2.2: Copilot Free -- бесплатный тариф

Отличная новость: с бесплатным аккаунтом GitHub ты уже получаешь
доступ к Copilot Free. Этого хватит для старта:

- **2000 автодополнений кода** в месяц
- **50 сообщений AI-агенту** в месяц

Активировать отдельно ничего не нужно -- Copilot Free включается
автоматически, как только ты войдёшь в GitHub из VS Code.

> **Советуем взять Copilot Pro ($10/мес) хотя бы на первый месяц.**
> В нём можно выбрать модель Claude, которая лучше всего
> справляется с ролью ментора. Ещё в Pro больше запросов
> к агенту и нет ограничений на автодополнения.
> Но и бесплатной версии хватит для старта.

Если решишь взять Pro:

1. Открой [github.com/settings/copilot](https://github.com/settings/copilot)
2. Нажми **"Start free trial"** или **"Upgrade to Pro"**
3. Введи данные карты и оформи подписку

<!-- screenshot: страница настроек Copilot с выбором тарифа -->

> **⚠️ Оплата из России**
>
> Российские карты напрямую не работают (GitHub использует Stripe).
> Варианты:
> - Виртуальные карты иностранных банков
>   (например, через сервисы-посредники)
> - Попросить друга/знакомого за рубежом
> - Криптокарты с поддержкой международных платежей
>
> Если оплатить не получается -- не переживай, бесплатного
> тарифа достаточно чтобы пройти весь курс.

### Шаг 2.3: Установка расширения

1. Открой VS Code
2. Нажми **Ctrl+Shift+X** -- откроется панель расширений слева
3. В строке поиска набери **"GitHub Copilot"**
4. Найди расширение от **GitHub** (обрати внимание на автора --
   именно GitHub, с галочкой подтверждения)
5. Нажми кнопку **"Install"** (Установить)
6. Подожди несколько секунд пока установится

Вместе с ним автоматически установится **GitHub Copilot Chat** --
это расширение для чата, через который мы и будем работать.

<!-- screenshot: поиск GitHub Copilot в расширениях VS Code -->

### Шаг 2.4: Вход в GitHub из VS Code

1. После установки расширения VS Code попросит войти в GitHub
2. Нажми **"Sign in to GitHub"** (Войти в GitHub)
3. Автоматически откроется браузер со страницей авторизации
4. Нажми **"Authorize Visual-Studio-Code"**
   (Разрешить Visual Studio Code)
5. Вернись в VS Code -- расширение Copilot должно быть активно

Если уведомление не появилось:
1. Нажми на иконку аккаунта (человечек) в нижнем левом углу VS Code
2. Нажми **"Sign in with GitHub to use GitHub Copilot"**

<!-- screenshot: уведомление о входе в GitHub -->
<!-- screenshot: страница авторизации в браузере -->

### Шаг 2.5: Включение Agent Mode

Agent Mode -- это режим, в котором Copilot не просто подсказывает,
а может сам писать код, создавать файлы и запускать команды.
Именно этот режим нам нужен.

1. Нажми **Ctrl+Shift+I** -- откроется панель Copilot Chat
   (или нажми на иконку чата Copilot в верхней панели)
2. В верхней части панели чата найди переключатель режима
3. Выбери **"Agent"** (вместо "Chat" или "Edit")

<!-- screenshot: панель Copilot Chat с переключателем режима Agent -->

### Шаг 2.6: Выбор модели Claude (только Copilot Pro)

Если ты оформил подписку Copilot Pro, можно выбрать модель Claude,
которая лучше всего подходит как ментор:

1. Открой Copilot Chat (**Ctrl+Shift+I**)
2. В нижней части панели чата найди поле ввода
3. Под полем ввода (или в его нижнем углу) есть выпадающий список
   с названием текущей модели (по умолчанию GPT-4o)
4. Нажми на него и выбери **"Claude Sonnet"** из списка

<!-- screenshot: выпадающий список выбора модели в Copilot Chat -->

> На бесплатном тарифе выбор модели недоступен -- но встроенная
> модель тоже хорошо справляется с задачами.

### Проверка

1. Открой Copilot Chat (**Ctrl+Shift+I**)
2. Убедись что выбран режим **Agent**
3. В поле ввода напиши: **Привет**
4. Нажми Enter
5. AI-помощник должен ответить приветствием

Если он ответил -- всё работает. Переходи к следующему шагу.

<!-- screenshot: первое сообщение в Copilot Chat -->

> **⚠️ Если что-то пошло не так**
>
> - **Расширение не находится в поиске** -- убедись что VS Code
>   обновлён до последней версии (Помощь -> Проверить наличие
>   обновлений)
> - **Copilot не активируется** -- проверь что ты вошёл в GitHub
>   (иконка аккаунта в нижнем левом углу VS Code должна показывать
>   твоё имя)
> - **Copilot не отвечает** -- проверь подключение к интернету,
>   перезапусти VS Code (закрой и открой заново)
> - **Нет режима Agent** -- обнови VS Code и расширение Copilot
>   до последней версии

## Шаг 3: Клонирование репозитория

**⏱ ~5 минут**

### Что это такое

"Репозиторий" -- это папка с проектом, которая хранится в интернете
на сайте GitHub. "Клонирование" -- это скачивание этой папки
на твой компьютер. Тебе нужно скачать проект-инструкцию, в котором
AI-помощник найдёт всё необходимое чтобы вести тебя по процессу.

### Способ А: через VS Code (если Git установлен)

1. Открой VS Code
2. Нажми **Ctrl+Shift+P** -- появится строка команд вверху экрана
3. Начни набирать **"Git: Clone"** и выбери эту команду из списка
4. В появившееся поле вставь ссылку:
   ```
   https://github.com/TODO/vibecoding-intro.git
   ```
5. Выбери папку на компьютере куда сохранить проект
   (например, `D:\Projects` или `Документы`)
6. Нажми **"Select as Repository Destination"**
   (Выбрать как место для репозитория)
7. Дождись скачивания
8. Появится уведомление "Открыть клонированный репозиторий?" --
   нажми **"Open"** (Открыть)

<!-- screenshot: команда Git Clone в VS Code -->
<!-- screenshot: ввод URL репозитория -->

### Способ Б: скачать ZIP (если Git не установлен)

Не знаешь что такое Git? Не установлен? Не проблема!
Просто скачай проект как обычный файл:

1. Открой в браузере страницу проекта на GitHub:
   `https://github.com/TODO/vibecoding-intro`
2. Найди зелёную кнопку **"Code"** (обычно справа сверху)
3. В выпадающем меню нажми **"Download ZIP"**
4. Дождись скачивания
5. Найди скачанный файл (обычно в папке "Загрузки")
6. Нажми правой кнопкой мыши -> **"Извлечь всё..."**
7. Выбери папку куда распаковать (например, `D:\Projects`)
8. Нажми **"Извлечь"**
9. Открой VS Code
10. Нажми **Файл -> Открыть папку** (или **File -> Open Folder**)
11. Найди распакованную папку `vibecoding-intro` и выбери её
12. Нажми **"Выбрать папку"**

<!-- screenshot: зелёная кнопка Code и пункт Download ZIP на GitHub -->
<!-- screenshot: распаковка ZIP-архива -->

> **⚠️ Если Git не установлен**
>
> Это нормально! Просто используй способ Б (скачай ZIP).
> AI-помощник поможет установить Git позже, когда он понадобится
> для сохранения твоей работы. Сейчас Git не нужен.

> **⚠️ Если VS Code предлагает "Trust the authors"**
>
> При открытии папки VS Code может спросить "Do you trust the authors?"
> (Доверяете ли вы авторам?). Нажми **"Yes, I trust the authors"** --
> это безопасно.

## Поехали!

Всё готово. Ты на старте. Теперь:

1. Убедись что в VS Code открыта папка **vibecoding-intro**
   (название папки видно в заголовке окна и в боковой панели слева)
2. Открой **Copilot Chat** (нажми **Ctrl+Shift+I** или на иконку
   чата Copilot в верхней панели)
3. Убедись что выбран режим **Agent**
4. Напиши:

   > **Привет**

Copilot прочитает инструкции проекта и начнёт вести тебя через процесс
создания сайта. Он спросит какой сайт ты хочешь, поможет установить
всё необходимое и проведёт от идеи до работающего сайта в интернете.

Просто следуй его подсказкам. Если что-то непонятно -- спрашивай.
AI-помощник терпеливый, он объяснит столько раз, сколько нужно.
