# Установка и первый запуск

> Всего 3 шага -- и Claude начнёт помогать

**⏱ Время: ~15 минут**

Тебе нужно установить всего две вещи: редактор кода и AI-помощника.
Всё остальное (Node.js, Git, Docker) Claude поможет поставить потом,
когда это понадобится. Не нужно ставить всё заранее.

---

## Шаг 1: Установка VS Code

**⏱ ~5 минут**

### Что это такое

VS Code -- это бесплатный редактор кода от Microsoft. Думай о нём
как о Word, только для кода. Здесь ты будешь видеть свой проект
и общаться с Claude.

### Установка

1. Открой в браузере сайт [code.visualstudio.com](https://code.visualstudio.com)
2. Нажми большую синюю кнопку **"Download for Windows"**
3. Дождись окончания скачивания (файл появится внизу браузера)
4. Открой скачанный файл двойным кликом
5. Появится окно установки. Нажми **"Далее"**
6. Прими лицензионное соглашение. Нажми **"Далее"**
7. На экране "Выберите дополнительные задачи":
   - Поставь галочку **"Add to PATH"** (добавить в PATH) -- это важно
   - Остальные галочки можно оставить как есть
8. Нажми **"Установить"**
9. Дождись окончания установки
10. Нажми **"Готово"**

<!-- screenshot: сайт VS Code с кнопкой Download -->
<!-- screenshot: экран установки с галочкой Add to PATH -->

### Первый запуск

1. Найди VS Code в меню "Пуск" (набери "VS Code" в поиске Windows)
2. Запусти его
3. Ты увидишь приветственную страницу -- это нормально, просто закрой её

<!-- screenshot: приветственная страница VS Code -->

> **⚠️ Если что-то пошло не так**
>
> - **VS Code не запускается** -- перезагрузи компьютер и попробуй снова
> - **Ошибка при установке** -- нажми правой кнопкой мыши на установщик,
>   выбери "Запуск от имени администратора"
> - **Не находится в меню "Пуск"** -- попробуй набрать полное название
>   "Visual Studio Code" (не путай с Visual Studio -- это другая программа)

### Русификация (необязательно)

Если хочешь, чтобы интерфейс VS Code был на русском:

1. Нажми **Ctrl+Shift+X** -- откроется панель расширений слева
2. В строке поиска набери **"Russian Language Pack"**
3. Найди расширение от Microsoft (с иконкой флага)
4. Нажми кнопку **"Install"** (Установить)
5. Появится уведомление внизу справа -- нажми **"Change Language and Restart"**
6. VS Code перезапустится на русском языке

<!-- screenshot: установка Russian Language Pack -->

> Это необязательный шаг. Многие разработчики работают с английским
> интерфейсом -- большинство инструкций и туториалов написаны
> на английском. Но если с русским комфортнее -- ставь без сомнений.

## Шаг 2: Установка Claude Code

**⏱ ~5 минут**

### Что это такое

Claude Code -- это AI-помощник прямо внутри VS Code. Ты пишешь ему
что хочешь на обычном русском языке -- он делает. Именно он будет вести
тебя через весь процесс создания сайта.

### Подписка Claude Pro

Claude Code требует подписку Claude Pro ($20/мес, ~1800 руб).
Это **единственная обязательная платная часть** всего курса.

1. Открой в браузере [claude.ai](https://claude.ai)
2. Нажми **"Sign up"** (Зарегистрироваться)
3. Создай аккаунт (можно через Google или email)
4. После входа нажми на своё имя в левом нижнем углу
5. Выбери **"Upgrade to Pro"**
6. Введи данные карты и оформи подписку

<!-- screenshot: страница подписки Claude Pro -->

> **⚠️ Если не получается оплатить**
>
> - Убедись что карта поддерживает международные платежи
> - Попробуй другую карту
> - Виртуальные карты тоже подходят

### Установка расширения

1. Открой VS Code
2. Нажми **Ctrl+Shift+X** -- откроется панель расширений слева
3. В строке поиска набери **"Claude Code"**
4. Найди расширение от **Anthropic** (обрати внимание на автора --
   именно Anthropic, не другие)
5. Нажми кнопку **"Install"** (Установить)
6. Подожди несколько секунд пока установится

<!-- screenshot: поиск Claude Code в расширениях VS Code -->

### Авторизация

1. После установки в левой боковой панели VS Code появится
   иконка Claude (логотип -- стилизованная буква)
2. Нажми на эту иконку
3. Откроется панель Claude Code
4. Нажми кнопку **"Sign in"** (Войти)
5. Автоматически откроется браузер с сайтом Claude
6. Войди в свой аккаунт (тот, где оформлена подписка Pro)
7. Нажми **"Allow"** (Разрешить) чтобы дать VS Code доступ
8. Вернись в VS Code -- панель Claude Code должна быть готова к работе

<!-- screenshot: кнопка Sign in в панели Claude Code -->
<!-- screenshot: разрешение доступа в браузере -->

### Проверка

1. В панели Claude Code внизу есть поле ввода
2. Напиши туда: **Привет**
3. Нажми Enter
4. Claude должен ответить приветствием

Если он ответил -- всё работает. Переходи к следующему шагу.

<!-- screenshot: первое сообщение Claude Code -->

> **⚠️ Если что-то пошло не так**
>
> - **Расширение не находится в поиске** -- убедись что VS Code
>   обновлён до последней версии (Помощь → Проверить наличие обновлений)
> - **Авторизация не работает** -- проверь что подписка Pro активна
>   на [claude.ai](https://claude.ai)
> - **Claude не отвечает** -- проверь подключение к интернету,
>   перезапусти VS Code (закрой и открой заново)
> - **Кнопка Sign in не появляется** -- попробуй перезапустить VS Code

## Шаг 3: Клонирование репозитория

**⏱ ~5 минут**

### Что это такое

"Репозиторий" -- это папка с проектом, которая хранится в интернете
на сайте GitHub. "Клонирование" -- это скачивание этой папки
на твой компьютер. Тебе нужно скачать проект-инструкцию, в котором
Claude найдёт всё необходимое чтобы вести тебя по процессу.

### Способ А: через VS Code (если Git установлен)

1. Открой VS Code
2. Нажми **Ctrl+Shift+P** -- появится строка команд вверху экрана
3. Начни набирать **"Git: Clone"** и выбери эту команду из списка
4. В появившееся поле вставь ссылку:
   ```
   https://github.com/TODO/vibecoding-intro.git
   ```
5. Выбери папку на компьютере куда сохранить проект
   (например, `D:\Projects` или `Документы`)
6. Нажми **"Select as Repository Destination"**
   (Выбрать как место для репозитория)
7. Дождись скачивания
8. Появится уведомление "Открыть клонированный репозиторий?" --
   нажми **"Open"** (Открыть)

<!-- screenshot: ввод URL репозитория -->

### Способ Б: скачать ZIP (если Git не установлен)

Не знаешь что такое Git? Не установлен? Не проблема!
Просто скачай проект как обычный файл:

1. Открой в браузере страницу проекта на GitHub:
   `https://github.com/TODO/vibecoding-intro`
2. Найди зелёную кнопку **"Code"** (обычно справа сверху)
3. В выпадающем меню нажми **"Download ZIP"**
4. Дождись скачивания
5. Найди скачанный файл (обычно в папке "Загрузки")
6. Нажми правой кнопкой мыши → **"Извлечь всё..."**
7. Выбери папку куда распаковать (например, `D:\Projects`)
8. Нажми **"Извлечь"**
9. Открой VS Code
10. Нажми **Файл → Открыть папку** (или **File → Open Folder**)
11. Найди распакованную папку `vibecoding-intro` и выбери её
12. Нажми **"Выбрать папку"**

<!-- screenshot: зелёная кнопка Code и пункт Download ZIP на GitHub -->
<!-- screenshot: распаковка ZIP-архива -->

> **⚠️ Если Git не установлен**
>
> Это нормально! Просто используй способ Б (скачай ZIP).
> Claude поможет установить Git позже, когда он понадобится
> для сохранения твоей работы. Сейчас Git не нужен.

> **⚠️ Если VS Code предлагает "Trust the authors"**
>
> При открытии папки VS Code может спросить "Do you trust the authors?"
> (Доверяете ли вы авторам?). Нажми **"Yes, I trust the authors"** --
> это безопасно.

## Поехали!

Всё готово. Ты на старте. Теперь:

1. Убедись что в VS Code открыта папка **vibecoding-intro**
   (название папки видно в заголовке окна и в боковой панели слева)
2. Открой панель **Claude Code** (нажми на иконку Claude
   в левой боковой панели)
3. Напиши Claude:

   > **Привет**

Claude прочитает инструкции проекта и начнёт вести тебя через процесс
создания сайта. Он спросит какой сайт ты хочешь, поможет установить
всё необходимое и проведёт от идеи до работающего сайта в интернете.

Просто следуй его подсказкам. Если что-то непонятно -- спрашивай.
Claude терпеливый, он объяснит столько раз, сколько нужно.
