## Шаг 1: Установка Cursor

**⏱ ~10 минут**

### Что это такое

Cursor -- это бесплатный редактор кода со встроенным
AI-помощником. Думай о нём как о Word, только для кода --
и с AI внутри, который умеет писать код за тебя.

Cursor -- это форк (улучшенная версия) VS Code от Microsoft.
Если ты слышал про VS Code -- Cursor выглядит и работает
точно так же, только AI уже встроен и не нужно ничего
дополнительно настраивать.

### Скачивание и установка

1. Открой в браузере сайт [cursor.com](https://cursor.com)
2. Нажми большую кнопку **"Download"**

<!-- screenshot: сайт cursor.com с кнопкой Download -->

**Windows:**

3. Открой скачанный файл (.exe) двойным кликом
4. Появится окно установки. Следуй инструкциям на экране
5. Если спросит про настройки -- оставь всё по умолчанию
6. Дождись окончания установки

**macOS:**

3. Открой скачанный файл (.dmg)
4. Перетащи **Cursor** в папку **Программы** (Applications)
5. Готово!

<!-- screenshot: установка Cursor -->

### Первый запуск и создание аккаунта

- **Windows:** найди Cursor в меню "Пуск"
  (набери "Cursor" в поиске)
- **macOS:** найди Cursor в Launchpad или в папке Программы

При первом запуске Cursor попросит создать аккаунт:

1. Нажми **"Sign Up"** (Зарегистрироваться)
2. Введи свой **email** и придумай пароль
   (или войди через Google/GitHub -- как удобнее)
3. Подтверди email если попросит
4. Готово -- ты в Cursor!

<!-- screenshot: экран регистрации Cursor -->

> Если Cursor предложит настройки ("Import from VS Code",
> выбор темы и т.д.) -- просто пролистай, выбирая что нравится,
> или оставь всё по умолчанию.

> **⚠️ Если что-то пошло не так**
>
> - **Cursor не запускается** -- перезагрузи компьютер
>   и попробуй снова
> - **Windows: ошибка при установке** -- нажми правой кнопкой
>   мыши на установщик, выбери "Запуск от имени администратора"
> - **macOS: "приложение повреждено"** -- открой Системные
>   настройки → Конфиденциальность и безопасность → нажми
>   "Всё равно открыть"

### Русификация (необязательно)

Если хочешь, чтобы интерфейс Cursor был на русском:

1. Нажми **Ctrl+Shift+X** -- откроется панель расширений слева
2. В строке поиска набери: **MS-CEINTL.vscode-language-pack-ru**
   (это точный ID расширения -- по нему находится всегда)
3. Найди расширение от Microsoft (с иконкой флага)
4. Нажми кнопку **"Install"** (Установить)
5. Появится уведомление внизу справа -- нажми
   **"Change Language and Restart"**
6. Cursor перезапустится на русском языке

<!-- screenshot: установка Russian Language Pack -->

> **⚠️ Если расширение не находится**
>
> Маркетплейс расширений может быть недоступен без VPN
> из России. Если поиск ничего не показывает:
> - Попробуй включить VPN и поискать снова
> - Или набери точный ID: `MS-CEINTL.vscode-language-pack-ru`
> - Русификация необязательна -- если не получается,
>   спокойно пропускай этот шаг

> Это необязательный шаг. Многие разработчики работают с английским
> интерфейсом -- большинство инструкций и туториалов написаны
> на английском. Но если с русским комфортнее -- ставь без сомнений.

### Шаг 1.1: Знакомство с AI-помощником

AI в Cursor уже встроен -- ничего дополнительно ставить не нужно.
Справа в окне Cursor ты видишь панель **чата** -- это и есть
твой AI-помощник. Именно через этот чат мы будем работать
на протяжении всего курса.

Чтобы начать новый разговор с AI-агентом, нажми
**Ctrl+Shift+L** (или кнопку **"New Agent"** на экране
приветствия). AI-агент умеет сам писать код, создавать
файлы и запускать команды -- тебе остаётся только
проверять результат в браузере.

Попробуй прямо сейчас:

1. Нажми **Ctrl+Shift+L** -- откроется новый чат с агентом
2. Напиши: **Привет**
3. Нажми Enter
4. AI-помощник должен ответить

Если ответил -- отлично, AI работает!

### Шаг 1.2: Cursor Free vs Pro

Cursor Free -- бесплатный тариф, который ты уже получил
при регистрации. Его хватит для старта:

- **Unlimited Auto mode** -- AI сам выбирает лучшую модель
  для каждого запроса. Бесплатно и без ограничений
- **50 premium-запросов** в месяц -- для ручного выбора модели
- **2000 автодополнений** кода в месяц

**Cursor Pro** стоит $20/мес и даёт больше:

- Больше premium-запросов
- Ручной выбор модели: Claude Sonnet, Claude Opus, GPT и др.
- Background-агенты (AI работает в фоне)

**Наша рекомендация:** начни с бесплатного тарифа.
Auto mode отлично справляется с задачами курса. Если
почувствуешь, что хочешь больше контроля над моделью --
перейди на Pro.

> **⚠️ Лимиты Free-тарифа кончаются быстро**
>
> При активном общении с AI бесплатные запросы могут
> закончиться за одно занятие. Не переживай -- это нормально.
> Несколько советов:
>
> - **Не увлекайся полировкой сайта.** Цель курса --
>   пройти весь путь от идеи до деплоя, а не создать
>   идеальный сайт мечты. Сайт можно улучшать потом
>   сколько угодно, а пока важнее пощупать все технологии
> - **Формулируй запросы чётко** -- один конкретный запрос
>   лучше десяти расплывчатых
> - **Если лимиты кончились** -- подожди до следующего
>   месяца или перейди на Pro
>
> **Альтернатива:** если лимитов Cursor совсем не хватает,
> попробуй [Google Antigravity](https://antigravity.google/download)
> -- бесплатная IDE от Google на базе Gemini 3 с щедрыми
> лимитами. Скачивается и работает так же как Cursor
> (тоже форк VS Code).
>
> **Нюанс:** Antigravity заблокирован для российских
> Google-аккаунтов (проверяется регион аккаунта, а не IP).
> Обычный VPN не помогает. Для доступа нужен Google-аккаунт
> с иностранным регионом. Если такой есть -- Antigravity
> отличный вариант с очень щедрыми бесплатными лимитами.

> **⚠️ Оплата Pro из России**
>
> Российские карты могут не работать напрямую.
> Варианты:
> - Виртуальные карты иностранных банков
>   (через сервисы-посредники)
> - Попросить друга/знакомого за рубежом
> - Криптокарты с поддержкой международных платежей
>
> Но бесплатного тарифа достаточно чтобы пройти весь курс.

> **⚠️ Если AI не отвечает**
>
> - Проверь подключение к интернету
> - Перезапусти Cursor (закрой и открой заново)
> - Убедись что ты вошёл в аккаунт (иконка аккаунта
>   в правом верхнем углу)

## Шаг 2: Клонирование репозитория

**⏱ ~5 минут**

### Что это такое

"Репозиторий" -- это папка с проектом, которая хранится в интернете
на сайте GitHub. "Клонирование" -- это скачивание этой папки
на твой компьютер. Тебе нужно скачать проект-инструкцию, в котором
AI-помощник найдёт всё необходимое чтобы вести тебя по процессу.

### Клонирование через Cursor

1. Открой Cursor
2. Нажми **Ctrl+Shift+P** -- появится строка команд вверху экрана
3. Начни набирать **"Git: Clone"** и выбери эту команду из списка
4. В появившееся поле вставь ссылку:
   ```
   https://github.com/slip-pro/vibecoding-intro
   ```
5. Выбери папку на компьютере куда сохранить проект
   (например, `D:\Projects` на Windows
   или `~/Projects` на macOS)
6. Нажми **"Select as Repository Destination"**
   (Выбрать как место для репозитория)
7. Дождись скачивания
8. Появится уведомление "Открыть клонированный репозиторий?" --
   нажми **"Open"** (Открыть)

<!-- screenshot: команда Git Clone в Cursor -->
<!-- screenshot: ввод URL репозитория -->

> **⚠️ Если Cursor говорит что Git не установлен**
>
> Cursor может попросить установить Git. Это бесплатная
> программа для работы с кодом (подробнее о ней будет позже).
> 1. Зайди на [git-scm.com](https://git-scm.com)
> 2. Скачай и установи (все настройки по умолчанию)
> 3. Перезапусти Cursor и попробуй снова

> **⚠️ Если Cursor предлагает "Trust the authors"**
>
> При открытии папки Cursor может спросить "Do you trust the authors?"
> (Доверяете ли вы авторам?). Нажми **"Yes, I trust the authors"** --
> это безопасно.

## Поехали!

Всё готово. Ты на старте. Теперь:

1. Убедись что в Cursor открыта папка **vibecoding-intro**
   (название папки видно в заголовке окна и в боковой панели слева)
2. Открой чат с AI-агентом (нажми **Ctrl+Shift+L**)
3. Напиши:

   > **Привет**

AI-помощник прочитает инструкции проекта и начнёт вести тебя
через процесс создания сайта. Он спросит какой сайт ты хочешь,
поможет установить всё необходимое и проведёт от идеи
до работающего сайта в интернете.

Просто следуй его подсказкам. Если что-то непонятно -- спрашивай.
AI-помощник терпеливый, он объяснит столько раз, сколько нужно.
