# Что такое вайб-кодинг

Ещё недавно, чтобы создать свой сайт, нужно было
месяцами (а то и годами) учиться программированию.
Разбираться в HTML, CSS, JavaScript, фреймворках,
серверах... Порог входа был огромным, и большинство
людей с классными идеями так и не доходили до
реализации.

Но в 2024-2025 годах всё изменилось. Искусственный
интеллект научился писать код. И писать его хорошо.
Теперь не нужно знать ни одного языка
программирования, чтобы создать настоящий,
работающий сайт.

**Вайб-кодинг** — это когда ты описываешь словами,
что хочешь получить, а AI пишет код за тебя. Термин
придумал Андрей Карпатый (Andrej Karpathy) — бывший
директор по AI в Tesla и один из основателей OpenAI.
Идея простая: ты «ловишь вайб», объясняешь что нужно
на человеческом языке, а машина превращает это
в работающий продукт.

---

## Почему это работает

**AI действительно умеет писать код** — и делает это
на уровне опытного разработчика. Современные модели
(такие как Claude и GPT) обучены на миллионах проектов
и знают все популярные технологии.

Думай об этом так: **ты — креативный директор,
AI — исполнитель**. Ты придумываешь, что должно быть
на сайте, какие цвета, какие секции, какой текст.
AI берёт твои идеи и превращает их в код.

Не нужно знать синтаксис. Не нужно помнить команды.
Достаточно уметь объяснить, что ты хочешь.

> Аналогия: представь, что ты заказываешь мебель
> у мастера. Ты говоришь: «Хочу книжную полку из
> светлого дерева, три яруса, чтобы поместилась
> в нишу 80 см». Мастер делает. Ты смотришь:
> «Отлично, но добавь ещё один ярус». Мастер
> добавляет. Так же работает вайб-кодинг — только
> мастер это AI, а полка — твой сайт.

Не волнуйся, если что-то непонятно — AI-помощник
объяснит по ходу дела. Буквально каждый термин будет
разобран простым языком.

## Что мы будем строить

Мы вместе создадим **лендинг** — одностраничный сайт.
Это может быть:
- Портфолио
- Визитка для бизнеса
- Страница проекта
- Персональный сайт

Результат будет:
- **Красивым** — современный дизайн с анимациями
- **Адаптивным** — отлично выглядит на телефоне,
  планшете и компьютере
- **Настоящим** — доступен в интернете по твоему
  собственному домену

## Карта пути

Вот весь путь от нуля до сайта в интернете.
Каждый шаг — это отдельная инструкция с подробными
объяснениями.

```
Шаг 0: Знакомство с Copilot     ⏱ ~10 мин
Шаг 1: Установка Node.js        ⏱ ~10 мин
Шаг 2: Создание проекта         ⏱ ~15 мин
Шаг 3: Разработка сайта         ⏱ ~60 мин
Шаг 4: Git и GitHub             ⏱ ~20 мин
Шаг 5: Docker                   ⏱ ~20 мин
Шаг 6: Домен и VPS              ⏱ ~30 мин
Шаг 7: Деплой                   ⏱ ~30 мин
─────────────────────────────────────────
Итого                           ⏱ ~3-4 часа
```

Не пугайся объёма! Каждый шаг разбит на маленькие
действия, и AI-помощник будет рядом на каждом из них.
К тому же, не обязательно проходить всё за один
раз — можно делать по одному-два шага в день.

> Подсказка: после Шага 3 у тебя уже будет готовый
> работающий сайт на твоём компьютере. Шаги 4-7 —
> это про то, чтобы показать его всему миру.

> Домен и сервер нужны только если хочешь опубликовать
> сайт в интернете.

## Другие AI-инструменты

В этом курсе мы используем **VS Code + GitHub Copilot** — это самый
доступный вариант (работает без VPN из России, есть бесплатный тариф).

Но вайб-кодинг работает с любым AI-инструментом. Если у тебя
уже есть другой — можешь использовать его, процесс будет
примерно таким же:

| Инструмент | Что это | Цена |
|------------|---------|------|
| [Cursor](https://cursor.com) | Редактор на базе VS Code с мощным AI | $20/мес |
| [Windsurf](https://windsurf.com) | AI-редактор с чистым интерфейсом | $15/мес |
| [Google Antigravity](https://developers.googleblog.com/build-with-google-antigravity-our-new-agentic-development-platform/) | IDE от Google (Gemini), на базе VS Code | Бесплатно |
| [Claude Code](https://docs.anthropic.com/en/docs/claude-code) | CLI-инструмент от Anthropic | Нужна подписка Claude |
| [OpenCode](https://opencode.ai) | Open-source AI-агент для терминала | Бесплатно |

> Если ты совсем новичок — рекомендуем VS Code + Copilot.
> Все инструкции курса написаны именно для него.
